Week 1

In the first week I covered functions, methods, variables, if/else statements and pretty much everything you need to start building simple progrms. Being practical and writing code as frequently as possible whether you understand the syntax or not is the best way to learn a programming language, grudually it will start to make sense. Python syntax is almsot identical to Ruby so this made things very easy for me. I think this part is comfortable for everybody when they first try a new language, the logic is almost universal when it comes to code.

Week 2

Week 3

Probably the most exhausasting of the week, the last leg of the course. HTML and CSS was introduced and we stepped into Web Development. Due to the linear syntax of HTML is was very easy to graps and familiarise however, like most people CSS was... annoying. Its syntax has an annoying curce with too many layers, unlike Python is is very customizable and can be off-putting. Once I understoof the genral logic in the syntax using tutorial and tools to build projects became easy. I found the best way to work with CSS is to not complicate things.
